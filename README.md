# Food Trends Understanding Customer Preferences
## Project Overview:

This project was developed as part of the Infosys Springboard Virtual Internship. It focuses on analyzing customer food ordering behaviour and preferences in the Food & Beverage (F&B) domain using data analytics and visualization techniques. The project transforms raw customer data into meaningful insights through interactive Power BI dashboards to support data-driven decision-making.

## Project Files:

Power BI dashboard files containing customer behavior analysis

Dataset used for customer food ordering behaviour analysis

Project documentation and reports

README file explaining project structure and insights

## Key Features:

Analysis of dominant food ordering platforms

Customer preference analysis (Veg vs Non-Veg, cuisine choices, taste, quality)

Ordering behaviour analysis across time slots and platforms

Delivery experience and service quality evaluation

Restaurant and cuisine performance insights

Interactive dashboards with filters and slicers for dynamic exploration

## Tools & Technologies:

### Power BI

Interactive dashboard creation

KPI cards, bar charts, column charts

Slicers and filters for City, Gender, and Monthly Income

Basic DAX measures such as AVERAGE and COUNT

### Microsoft Excel

Data entry, cleaning, and validation

Sorting, filtering, and consistency checks

Dataset understanding before Power BI import
